Privacy

The short version: NovaLabs doesn't track you, doesn't run ads, and doesn't sell anything about you. Here's the longer version.

Last updated: June 2026

Who we are

NovaLabs is an independent hardware and software lab run by dash, based in the United States. This policy covers the NovaLabs website (novalabs.app and its pages), the small web tools we host, and the RPCortex operating system you flash onto your own hardware.

The website

  • No analytics, no cookies, no tracking pixels. The site is static — there's nothing watching what you click.
  • Standard server logs. Like any website, the server that delivers these pages may record routine request data (such as your IP address and browser type) for security and reliability. We don't build profiles from it.
  • Links out. Some buttons take you to GitHub, Instagram, or Discord. Once you're there, those companies' own privacy policies apply.

The tools we host

ytdl-UI (yt.novalabs.app) takes a link you paste and fetches that media for you to download. It doesn't require an account, and it isn't built to keep a history of what you grab. Please only use it for content you have the right to download.

RPCortex & the Nova D1

RPCortex runs entirely on your device. Things like your account, password hash, and saved WiFi credentials are stored locally on the device's own flash — none of it is sent to NovaLabs. A couple of honest notes for owners:

  • WiFi passwords are stored in plaintext on-device. Microcontrollers generally have no secure enclave, so treat the device like a key — anyone with physical access can read its flash.
  • Network features reach out only when you use them. Commands like wget, ntp sync, or pkg install connect directly to the service you point them at; NovaLabs isn't in the middle.
  • OTA updates fetch release files from NovaLabs' servers when you run them. That's a normal web request — no device identifiers are required.

Children

NovaLabs' projects aren't directed at children under 13, and we don't knowingly collect information from them.

Changes

If this policy changes, the date above changes with it. Material changes will be noted on this page.

Contact

Questions about privacy? Email support@novalabs.app or see the contact page.
